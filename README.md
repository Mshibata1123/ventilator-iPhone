# 人工呼吸器アラーム対応トレーニングアプリ

臨床工学技士・看護師・研修医向けの人工呼吸器アラーム対応シミュレーショントレーニングアプリケーションです。

## 概要

このアプリケーションは、人工呼吸器のアラーム発生時の対応手順を学習できるインタラクティブなトレーニングツールです。実際の動画を確認しながら、適切な対応を選択する分岐型シナリオ形式で学習できます。

## 特徴

- 📱 **モバイル対応**: iPhone 12を含むスマートフォンで快適に使用可能
- 🎥 **実践的な学習**: 実際のアラーム動画を見ながら学習
- 🎮 **インタラクティブ**: 分岐型シナリオで実践的なトレーニング
- 📊 **学習記録**: 学習履歴を保存・確認可能
- 🎨 **モダンUI**: 美しく使いやすいインターフェース

## 使用方法

### GitHub Pagesでの公開

1. GitHubリポジトリを作成
2. すべてのファイルをリポジトリにアップロード
3. リポジトリの Settings > Pages で GitHub Pages を有効化
4. Source を `main` ブランチに設定
5. 表示されるURLからアクセス

### ローカルでの実行

1. このリポジトリをクローン
2. `index.html` をブラウザで開く
